Addendum

I tried to keep everything simple for that little JS program, using mostly Jquerry and event delegation whenever necessary. I placed all the cells of my game in a div element and formatted the board with a CSS grid. For the cells of the game, I used event delegation on the parent element (the div) of those cells. With that, I was able to know which cell was clicked using the indexing function of jQuery. With my index and some simple math (i+5. i-5, i-1, i+1), I was able to implicitly assign a position around the clicked element (above the clicked element, below the clicked element, on the right or left of the clicked element). From there, everything was simple: the program went through all the neighboring elements and set them in the isOff class if they were in the isOn class or the opposite if they were in the inOff class. I chose to display information to make everything seem very interactive. I displayed the minimum moves possible, which is random and is set before the program even starts. I also displayed the player's current move number. I also thought it would be nice to display a timer, and that has given me more insight into how time intervals work in Js.
